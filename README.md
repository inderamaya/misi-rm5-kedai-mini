# Misi RM5: Kedai Mini Ajaib

Aplikasi/game pembelajaran interaktif untuk **MBPK Masalah Pembelajaran Tahap 2**. Konsep reka bentuk menggunakan gaya **retro platformer original** yang ceria, tanpa menggunakan aset rasmi Super Mario/Nintendo.

## Objektif Pembelajaran

Murid dapat:
1. Mengenal nilai RM5.
2. Menyusun langkah membeli barang menggunakan Peta Alir.
3. Memilih barang berdasarkan bajet RM5.
4. Membandingkan pilihan barang berdasarkan harga, keperluan dan manfaat.
5. Memberi sebab mudah bagi pilihan barang.

## Ciri Mesra MBPK

- Arahan ringkas dan berulang.
- Butang besar sesuai untuk sentuhan.
- Tiada pemasa untuk mengurangkan tekanan.
- Audio arahan menggunakan Web Speech API.
- Visual, emoji dan simbol sebagai sokongan.
- Mod teks besar, kontras tinggi dan kurangkan gerakan.
- Maklum balas positif segera.
- Progress disimpan secara localStorage, tanpa pangkalan data.

## Teknologi

- React + Vite
- CSS responsif
- Web Speech API
- LocalStorage
- Sesuai deploy ke Vercel

## Jalankan secara lokal

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
npm run preview
```

## Deploy ke Vercel

Framework preset: Vite  
Build command: `npm run build`  
Output directory: `dist`

## Nota Hak Cipta

Projek ini tidak menggunakan watak, logo, audio atau grafik rasmi Super Mario/Nintendo. Tema visual ialah retro platformer generik yang original dan pendidikan.
